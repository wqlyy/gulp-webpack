# webpack&&gulp
## gulp与webpack的相结合，支持 sass less ES6 监听文件改动自动刷新
### 目录说明
#### *1.src开发目录，就是编写代码的地方
#### *2.dist发布（生产）目录，里面文件是经压缩，丑化，合并，构建好的目录
#### *3.‘.babelrc’是配置es6的。
#### *4.‘.jshintrc’是一个叫gulp-jshint插件对应的配置文件，这个是用来检查js书写规范和错误的，配置是配置他的规则。
#### *5.浏览器刷新需要安装“livereload”插件
#### *6.所有构建后的代码全部在dist文件下。
